# Apress::Changelogger

Changelogger

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'apress-changelogger'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install apress-changelogger

## Usage

```ruby
Apress::ChangeLogger.new.log_changes
```
